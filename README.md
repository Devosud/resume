# Resume

# Prerequisites

- [Docker](https://docs.docker.com/install/)
- [Docker compose](https://docs.docker.com/compose/install/)

# Build and launch

```shell
$ cd resume/
$ docker-compose down && docker-compose build && docker-compose up
```

Then, go to `http://localhost:4200/` on your web browser
